# 京急列車位置情報

## URL
https://tid.keikyu.co.jp/data/local/zaisen.json

## JSON
### UP

```
0: [
    "dt": [
        0: {
            "yy": 年,
            "mt": 月,
            "dy": 日,
            "hh": 時,
            "mm": 分,
            "ss": 秒
        }
    ],
    "st": ?
]
```
